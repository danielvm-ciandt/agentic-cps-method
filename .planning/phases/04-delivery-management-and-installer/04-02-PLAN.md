Resume anyway or discard?" Ask: "Type 'resume' to continue or 'discard' to delete the pause state." If user types `discard`: delete `{planningDir}/continue-here.md` and HALT.

**Step 3: Verify git branch** ŌĆö run `git branch --show-current`. If current branch does not match `branch` from continue-here.md: output warning "Current branch ({current}) does not match paused branch ({saved_branch}). Switch with: `git checkout {saved_branch}`". Ask: "Continue on current branch? (yes / no)". If no: HALT.

```
**acps-resume ŌĆö State Restored**

Paused at:   {paused_at}

## Open Tasks

{open tasks list from continue-here.md}

Pick up where you left off. Run `acps-{loop_phase}` to continue.
```

**Step 5: Offer cleanup** ŌĆö ask: "Delete continue-here.md now that you have resumed? (yes / no)". If yes: delete `{planningDir}/continue-here.md`. Output: "continue-here.md removed."
