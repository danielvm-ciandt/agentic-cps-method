**Step 3: Compute scope balance** — scan `{planningDir}/change-log.md` CR table for all BCP Delta and FP Delta values. Sum them to get `current_bcp` = `baseline_bcp + sum(BCP Delta)` and `current_fp` = `baseline_fp + sum(FP Delta)`. If change-log.md is missing, output "Scope: not tracked (run acps-gate to set baseline)".

**Step 4: Output status dashboard** — display formatted status block:

```
**acps-status**

Phase:       Setup / Production Flow / Value Activation (derived from backlog.md)
Iteration:   iter-{N} (from git branch)
Active epic: {epic-id} — {epic title} (from backlog.md)

Scope balance:
  BCP baseline:  {baseline_bcp}
  BCP current:   {current_bcp}  (delta: {current_bcp - baseline_bcp})
  FP baseline:   {baseline_fp}
  FP current:    {current_fp}   (delta: {current_fp - baseline_fp})
```
